# Treasure Hunt

- Descrição:
Jogo 2D desenvolvido em Python com Pygame, onde o jogador precisa explorar um taboleiro, evitar obstáculos e encontrar os tesouros escondidos. Projeto final da cadeira de Introdução a Programação

- Tecnologias Utilizadas:
1- Python 3
2- Pygame 2.6.1
